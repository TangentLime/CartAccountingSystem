# ServerClientSystem

ServerClientSystem is a pilot plan for a visual accounting system for carts carrying production materials.

## Installation

Use whatever you have to to install ServerClientSystem (if it ever gets published to a repo)

## Description

Contains code for a python server using flask that receives input from scanners (clients) and modifies a local database with the updates. The scanners use april tags to recognize carts and update their location based on where the scanners are located.
